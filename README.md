# SC-OtwCerdas

This is the repository for AI course project from OtwCerdas

## Installation

1. Download git
2. Download python
3. Download pip
4. Add environment using command below (CMIIW)

```bash
py -m venv env
```
5. Activate environment

```bash
env\Scripts\activate
```
6. Install Django
```bash
pip install django
```
7. Create a new folder
8. Clone or remote this repository
